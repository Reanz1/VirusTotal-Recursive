# VirusTotal-Recursive


This program helps users quickly and safely identify malicious files within a folder by leveraging VirusTotal’s multi-engine scanning service. It saves time by automatically checking known files, uploading unknown ones for analysis, and presenting clear, organized results with direct access to detailed reports—reducing manual effort and improving overall system security awareness.
